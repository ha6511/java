-. class  는 자바 파일 저장하는 단위 
-. API  는 회사에서 제공하는 clss 파일을 모아 놓은 것 
-. Ex01 는 clss  파일 의 이름 
-. class 파일의 이름은 무조건 첫글자는 대문자 
-. 무조건 파일의 이름과 클래스이름은 같아야 한다.
-. 저장이름 = 클래스이름 = 파일이름 


System.out.println("자바의 꿈")
-. 웹 브라우자 에서 즐겨찾기 AIP  호추

-. 왼쪽 하단 클래스 창에서 모든 클래스가 나타맘
-. ctrl+ f => 찾기 대화상자 호출
-. 해당 클래스 입력 하여 찾는다 ( Sysyem)
-. in =>  표준입력장치 키보드를 뜻한다.
-. out => 표준출력장치 모니터를 뜻한다.
-. err =>  표준오류출력장치 모니터를 뜻한다.

xxxxx([내용]) :  
자바에서 ()는 무조건 메소드이다. 
메소드란 기능, 동작, 작동, 실행을 의미한다.
메소드는 클래스 안에 존재한다.


main() 
-. JVM 이 해석하고 실행할때 제일 처음 호출해서 사용하는 메소드
-. 해당 프로그램에서 오직 하나만 존재 (시작 위치)
-. 없어도 안되고 여러개 있어도 안됨 

데이터 타입 ( 자료형 )
-. 처리할 수 있는 자료의 형태 
-. boolean(참, 거짓), 문자, 숫자(정수:소숫점이 없는 수, 실수:소숫점이 있는 수)

데이터를 저장하는 저장공간 을 변수, 상수 

변수 : 데이터를 변경할 수 있음 
        변수의 이름은 소문자로 시작한다.
        가장 최근의 정보 하나만 기억한다.
상수 : 데이터를 변경할 수 없음 
        상수의 이를은 모든 글자를 대문자로 사용.


변수나 상수를 호출하면 가지고 있는 데이터가 나온다
